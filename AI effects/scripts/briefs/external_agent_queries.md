# External AI Agent Queries — Literature Sweep

## Purpose
These queries are designed to be submitted to external AI tools (Perplexity, ChatGPT, Gemini, etc.)
to generate mini literature reviews per theme. Their output is treated as **leads only** —
every source mentioned must be independently verified before use.

## Instructions for saving responses
- Save each response in: `tmp/external_ai_responses/`
- Filename: `[theme]_[tool]_[date].txt`  e.g. `platform_traffic_perplexity_2026-03-17.txt`
- Note which tool produced it and when at the top of the file

## How to use the output
1. Extract every source title/author/year the AI mentions
2. Search for each independently (DOI, Semantic Scholar, arXiv)
3. If the source exists and is real, add it to `sources.txt` as status: candidate
4. Discard any source you cannot independently verify
